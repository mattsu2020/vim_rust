vimspell
========

This Rust tool spell checks a file and generates the syntax statements
necessary to highlight the errors in Vim.  It wraps the Hunspell library.

Build the project with `cargo build --release` and place the resulting
`vimspell` binary somewhere in your `$PATH` (for example, in this
`runtime/tools` directory).  Add convenient mappings in your `.vimrc`:

        noremap <F8> :so `vimspell %`<CR><CR>
        noremap <F7> :syntax clear SpellErrors<CR>

A local dictionary file can be provided with the `--dict` option or the
`LOCAL_DICT` environment variable.
